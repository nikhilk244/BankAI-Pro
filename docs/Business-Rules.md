# Business Rules

## Business Rules

| Rule ID | Business Rule |
|----------|---------------|
| BR-001 | One customer can own multiple bank accounts. |
| BR-002 | Every account belongs to exactly one customer. |
| BR-003 | Account numbers are generated by the system and cannot be changed. |
| BR-004 | Aadhaar number cannot be modified after KYC verification. |
| BR-005 | PAN number must be unique for each customer. |
| BR-006 | Email address must be unique. |
| BR-007 | Mobile number must be unique. |
| BR-008 | One account can have multiple transactions. |
| BR-009 | One customer can apply for multiple loans. |
| BR-010 | Every money transfer creates both a debit transaction and a credit transaction. |
| BR-011 | A customer cannot transfer more money than the available account balance unless an overdraft facility is enabled. |
| BR-012 | Every transaction must store the transaction date and time for auditing. |
