# 🏦 BankAI-Pro

## Overview
BankAI-Pro is an AI-powered digital banking application built using Spring Boot and Java.

The project demonstrates enterprise-level banking features along with AI-powered fraud detection, loan eligibility prediction, and intelligent banking assistance.

## Technology Stack

- Java 25
- Spring Boot
- Maven
- MySQL
- Hibernate / JPA
- REST API
- Git & GitHub
- AI Integration (Planned)

## Project Status

🚧 Currently under development.

This project is being built from scratch following professional software engineering practices.
